start

(
  • options
)

Defined in src/core.js:205

Startup Brass Monkey

Parameters:

  • options Object

    Options for initializing Brass Monkey.

    • name String

      The name of your game. This is what is displayed on the phone when you pick what to connect to.

      Example: "Monkey Golf"
    • appID String

      The unique identifier of your Game/App. You must generate this through the developer admin interface.

    • [maxPlayers=1] Number optional

      The number of players that can play your game.

    • [swfURL] URL optional

      The location of the SWF file that implements the SDK when there a browser doesn't support Websockets. 

      If not provided the matching version on the Brass Monkey CDN is used.

showslotcolor

Defined in src/core.js:338

Event called when the SDK has received it's assigned slot index/color.

This is the unique color that is displayed in top right of the console, in your game, and on the device list in the controller app. It's used to help users disambiguate which console/game they want to connect to, if there is more than one on their network.

Event Payload:

  • type String

    Which event this is. Ie. "showslotcolor"

  • slot Number

    Unique identifier to your network. (Each instance of Brass Monkey running get it's own slot number assigned.)

  • color String

    Color to display on the screen. (CSS Style Hexidecimal)

    Example:

    "#ff0000"
